Lesson 4 - CSS basics del 2

We already know how to select elements in CSS and manage their size, colour or visibility. In this lesson, we’ll learn how to position them so they look better and the breaks between them are sized correctly. To achieve this, we’ll learn how to separate adjacent elements with borders. Borders allow us to see where our elements end. Even in our tables, it wasn’t that clear thus far. However, we don’t always want our elements to have a visible border. Fortunately, we can use margins and paddings to manage the breaks between the elements, and they are entirely invisible. We’ll also learn how to position an element on our website. To achieve this, we’ll learn how to change the float and position properties. In the end, we’ll also create some basic animations to show that CSS can help us create dynamic websites without JavaScript.
