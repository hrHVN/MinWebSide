The Task

In this lesson, we learned about standard Bootstrap basics. In this task, you need to create a website that contains:

  • Three text paragraphs that behave like h1, h2 and h3 headings. Using Bootstrap classes, add padding-bottom of the medium size to them.
  • Create a list group containing three buttons. On click, each button should change colour to green.
  • Create a figure with an image and caption at the end. The caption should contain at least one Bootstrap icon.
