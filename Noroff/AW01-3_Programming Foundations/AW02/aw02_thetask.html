In this lesson, we learned some crucial concepts of Object-oriented Programming. 
    We learned about classes, encapsulation and inheritance.
    
    In this task, you have to create a menu for a restaurant. 
    Every dish has a name and price. Our restaurant serves pizza, pancakes and pasta. 
    In addition, pizza has an array of ingredients, and pancakes have the Boolean sweet property, which defines whether pancakes should be served sweet. 
    Pasta has the Boolean property isVegetarian, which defines whether it contains meat.
    
    Create a model in Object-oriented Programming simulating the menu. 
    Think about which properties should be private and which not.
