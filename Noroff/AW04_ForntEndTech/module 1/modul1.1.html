Lesson 1 - HTML basics

HTML stands for HyperText Markup Language. It's the set of markup codes and symbols inserted into a file intended for display on the Internet. The markup tells web browsers how to display a page's words and images. HTML is the primary tool for front-end developers. Thanks to it, we can create various elements on our website. After creating them, we can use Cascading Style Sheets to change their location and improve their appearance. We can also bind JavaScript code to them to make them interactive. In this lesson, we'll create a simple website using only HTML. We'll learn about elements such as headings, paragraphs and links to make our text display correctly. This lesson will help us understand more complex elements of HTML, which we'll learn about later.
